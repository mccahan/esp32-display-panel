# ESP32 Display Manager

Touchscreen home device built on the Guition ESP32-S3-4848S040 and the [esp32-display-claude-base](https://github.com/mccahan/esp32-display-claude-base) tool.

## Hardware

**Guition ESP32-S3-4848S040**
- 480x480 IPS display (ST7701 controller)
- Capacitive touch (GT911)
- ESP32-S3 with 8MB PSRAM
- 16MB Flash
